# Руководство по ручному тестированию

[English](./MANUAL_TESTING.md) · **Русский**

Этот документ описывает, как вручную проверить расширение AI Brief Decoder в Chrome.

## Предварительные требования

- Установлен Google Chrome (или Chromium).
- Расширение собрано:

  ```bash
  npm install
  npm run build
  ```

  В результате создаётся загружаемый каталог расширения `.output/chrome-mv3/`.

## Загрузка распакованного расширения

1. Откройте Chrome.
2. Перейдите по адресу `chrome://extensions/`.
3. Включите **Режим разработчика** переключателем в правом верхнем углу.
4. Нажмите **Загрузить распакованное расширение**.
5. Выберите каталог `.output/chrome-mv3/` внутри этого проекта.
6. Убедитесь, что расширение появилось в списке под названием **AI Brief Decoder Extension**.

## Проверка попапа

1. Нажмите на значок расширений на панели инструментов Chrome.
2. Нажмите на значок **AI Brief Decoder Extension**.
3. Убедитесь, что попап открывается и отображает текст **AI Brief Decoder Extension**.

## Проверка консоли на ошибки

1. Кликните правой кнопкой мыши внутри попапа и выберите **Просмотреть код** (Inspect).
2. В открывшемся окне DevTools перейдите на вкладку **Console** (Консоль).
3. Убедитесь, что сообщений об ошибках нет.

## Ожидаемый результат

- Расширение загружается без ошибок Chrome на странице `chrome://extensions/`.
- Попап открывается и отображается корректно.
- В консоли попапа нет ошибок.
