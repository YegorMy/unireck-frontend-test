# Расширение AI Brief Decoder

[English](./README.md) · **Русский**

Минимальное расширение для Chrome на основе [WXT](https://wxt.dev/), React и TypeScript.

## Менеджер пакетов

В проекте используется **npm**.

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Команда запускает WXT в режиме разработки и собирает распакованное расширение.

## Ручное тестирование

Пошаговая инструкция по загрузке собранного расширения в Chrome и проверке попапа — в [MANUAL_TESTING.ru.md](./MANUAL_TESTING.ru.md).

## Сборка

```bash
npm run build
```

## Проверка типов

```bash
npm run typecheck
```

## Структура проекта

- `entrypoints/popup/` — HTML попапа, точка монтирования React и корневой компонент.
- `components/` — заготовка под UI-компоненты функциональности.
- `lib/` — заготовка под типизированный клиент API.
